Démarrage

Ouvrez le menu "Fichier" et sélectionnez "Ouvrir dossier...". Sélectionnez tous les dossiers ou volume que vous voulez analyser. Une fenêtre va s'ouvrir en affichant "Chargement de bla bla". Soyez patient si vous avez sélectionné un gros dossier ou tout un volume avec beaucoup de fichiers et dossiers.
Dès que le chargement est terminé (la fenêtre de suivi a disparu) la fenêtre principale va s'ouvrir et vous voyez sur la moitié gauche une vue du style Finder du dossier/volume sélectionné et sur la moitié droite une vue graphique :

(1) Cette vue de style Finder montre les fichiers et dossiers avec leur tailles.
(2) La carte montre les fichiers et dossiers d'une façon qui met en évidence quels fichiers et dossiers occupent le plus de place. La sélection dans la liste des fichiers/dossiers et la carte est synchronisée. (Voir (4) pour la signification des différentes couleurs.)
(3) Ici quelques informations sur le fichier pointé sur la carte.
(4) Les statistiques par type de fichier donnent le nombre de fichiers pour un type spécifique et la somme de leur tailles. La liste est triée par taille en ordre décroissant. Chaque type de fichier a une couleur qui est aussi utilisée dans la carte (2). La même couleur est utilisée pour chaque type pour tous les dossiers ouverts (voir ci-dessous).

Dans le menu "Affichage" vous pouvez choisir si vous préférez voir le contenu des paquets, vous pouvez zoomer dans les dossiers et quelques autres choses qui aideront à analyser l'utilisation de l'espace disque.

Un mot sur l'utilisation des couleurs

Disk Inventory X a 12 couleurs prédéfinies qui sont affectées au 12 "plus gros types" (l'idée derrière ça est qu'il est de plus en plus difficile de distinguer les couleurs si elles sont plus de 12).
Le reste des types utilisent la couleur grise. Cette affectation de couleur est faite pour tous les dossiers ouverts (l'affectation de couleur est partagée) : les fichiers du même type ont la même couleur dans tous les dossiers ouverts.
Donc si vous chargez le premier dossier, les 12 couleurs peuvent être affectées à des types (si le dossier contient des fichiers d'au moins 12 types différents). Si vous ouvrez ensuite un deuxième dossier avec des types totalement différents, il ne reste plus de couleur et tous les types du deuxième dossier utilisent la couleur grise.
Le seul moyen de refaire les affectations est actuellement de redémarrer DIX.
